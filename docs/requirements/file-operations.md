# R6 — File operations

**Status:** Prototype scope

## Summary

Act on files directly from the UI.

## Behavior

- **Delete files** through the UI — writes back to the drive and updates the catalog.

> Drives are assumed **writable** in the prototype. Read-only media is a [future](./README.md#future) consideration.
